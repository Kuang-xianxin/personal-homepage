# 个人主页（Vue 3 + Vite）

基于 **Vue 3 + Vite** 搭建的简洁个人主页模板，你可以在其中修改名字、简介、技能、项目和联系方式。

## 安装与运行

```bash
cd "d:\cursor项目文件\个人主页"

# 安装依赖
npm install

# 启动开发环境
npm run dev
```

浏览器会自动打开（或手动访问）`http://localhost:5173/`。

## 可以修改的地方

- `src/App.vue` 中：
  - 顶部导航标题 `MySpace`
  - 你的名字、简介、技能描述、项目介绍
  - 联系方式（邮箱、GitHub 链接等）

修改完保存，页面会自动热更新。
